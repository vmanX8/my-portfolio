# Vangelis Manouhos - Front-End Portfolio

Simple portfolio site built with React, TypeScript, and Vite. Styled with plain CSS.

## Tech
- React
- TypeScript
- Vite
- CSS

## Run locally
```bash
git clone https://github.com/vmanX8/my-portfolio.git
cd my-portfolio
npm install
npm run dev
```

Open http://localhost:5173.
